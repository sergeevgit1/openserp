# Установка n8n-nodes-openserp

## Способ 1: Через интерфейс n8n (рекомендуется)

1. Откройте n8n
2. Перейдите в **Settings → Community Nodes**
3. Нажмите **"Install a community node"**
4. Введите: `n8n-nodes-openserp@1.0.8`
5. Нажмите **"Install"**
6. Перезапустите n8n, если потребуется

## Способ 2: Через командную строку

```bash
# В директории вашего n8n проекта
npm install n8n-nodes-openserp@1.0.8

# Или глобально
npm install -g n8n-nodes-openserp@1.0.8
```

## Способ 3: Через Docker

Добавьте в ваш Dockerfile:

```dockerfile
RUN npm install n8n-nodes-openserp@1.0.8
```

## Проверка установки

После установки узлы OpenSERP должны появиться в списке доступных узлов n8n.

## Требования

- n8n версии 0.200.0 или выше
- Node.js версии 16 или выше
- Доступ к API OpenSERP (если требуется)

## Устранение неполадок

Если узел не появляется после установки:

1. Перезапустите n8n
2. Проверьте консоль на наличие ошибок
3. Убедитесь, что версия n8n соответствует требованиям
4. Попробуйте переустановить пакет:

```bash
npm uninstall n8n-nodes-openserp
npm install n8n-nodes-openserp@1.0.8
```
